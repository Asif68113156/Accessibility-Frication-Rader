# ♿ Accessibility Friction Reader

A Django-based web application designed to identify accessibility barriers on web pages and help developers create more inclusive and user-friendly websites. The project analyzes web content, detects common accessibility issues, and provides insights to improve usability and compliance with accessibility standards.

---

## 📖 Overview

The Accessibility Friction Reader is built to assist developers, designers, and organizations in evaluating the accessibility of their websites. It helps identify usability challenges that may affect users with disabilities and offers recommendations for improving the overall user experience.

---

## ✨ Features

* 🔍 Analyze website accessibility
* ♿ Detect common accessibility issues
* 📊 Generate accessibility reports

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/accessibility-friction-reader.git
```

### Navigate to the Project

```bash
cd accessibility-friction-reader
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py migrate
```

### Run the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
